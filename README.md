# IPSBarber 

A full-stack barber booking and grooming marketplace platform built using Flutter, ASP.NET Core, PostgreSQL, and Supabase.

## 🌍 Live Deployment

### Frontend (Flutter Web)

Frontend URL: [https://ipsbarber.vercel.app](https://ipsbarber.vercel.app)

### Backend API (ASP.NET Core)

Backend URL: [https://ipsbarber.onrender.com](https://ipsbarber.onrender.com)

### Swagger API Docs

Swagger URL: [https://ipsbarber.onrender.com/swagger](https://ipsbarber.onrender.com/swagger)

# ⚙️ Backend Setup

## Navigate to backend

```bash
cd backend/KasiCuts.Api
```

## Install packages

```bash
dotnet restore
```

## Run migrations

```bash
dotnet ef database update
```

## Run API

```bash
dotnet run
```

---

# ⚙️ Frontend Setup

## Navigate to frontend

```bash
cd frontend/kasicuts_app
```

## Install dependencies

```bash
flutter pub get
```

## Run Flutter app

```bash
flutter run -d chrome
```
